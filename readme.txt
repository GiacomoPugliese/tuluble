Tuluble is a digital interface for luluble to manage inventory and communication with clients. This will be meant for internal use by employees.
Some of the functions included in this tool will be:
- inventory database
- formulas database
- ratio calculator

Technologies demonstrated: 
- OAuth --> to handle login and authorization
- MySQL --> for the database
- NodeJS --> to access database information
- HTML, CSS, JS --> front-end and back-end; display database
- AWS Cloud Server --> to host website
